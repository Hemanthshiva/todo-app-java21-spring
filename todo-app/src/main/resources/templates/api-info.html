Todo API Endpoints

This application primarily uses MVC controllers to serve HTML views rather than REST API endpoints. However, you can interact with the Todo functionality through the following URLs:

URL HTTP Method Description
/list-todos GET Lists all todos for the logged-in user
/add-todo GET Shows the page to add a new todo
/add-todo POST Adds a new todo
/todos/{id} DELETE Deletes a todo with the specified ID
/todos/{id} GET Shows the page to update (edit) a todo with the specified ID
/todos/{id} PUT Updates a todo with the specified ID

Note

The application is running on port 8091, so you can access it at http://localhost:8091.

The Swagger UI can be accessed at http://localhost:8091/swagger-ui/index.html.
